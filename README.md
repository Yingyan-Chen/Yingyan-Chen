## How I Work

I try to keep each learning step evidence-based:

1. Read the paper or repository.
2. Identify the smallest runnable unit.
3. Write or reproduce a minimal implementation.
4. Add tests, logs, or experiment records.
5. Summarize what changed in my understanding.

我的训练方式尽量以证据为核心：

1. 阅读论文或代码仓库。
2. 找到最小可运行单元。
3. 写出或复现最小实现。
4. 补充测试、日志或实验记录。
5. 总结理解上发生了什么变化。
